## 核心算法详解

### 1. 结构分析算法

#### 分型识别
```python
- 顶分型：中间K线高点最高，低点最高
- 底分型：中间K线低点最低，高点最低
```

#### 笔识别
```python
- 连接相邻的顶底分型
- 顶底分型之间至少有一根非共用K线
- 向上笔：从底到顶
- 向下笔：从顶到底
```

#### 线段识别
```python
- 向上线段：高点不断创新高，至少由三笔构成
- 向下线段：低点不断创新低，至少由三笔构成
```

#### 中枢识别
```python
- 至少由三段构成
- ZG（中枢上沿）：所有段高点最低值
- ZD（中枢下沿）：所有段低点最高值
- GG（中枢高点）：所有段高点最高值
- DD（中枢低点）：所有段低点最低值
```

### 2. 动力学分析算法

#### MACD计算
```python
- DIF = EMA(12) - EMA(26)
- DEA = EMA(DIF, 9)
- MACD = (DIF - DEA) * 2
```

#### 背驰识别
```python
- 顶背驰：价格创新高，MACD力度减弱
- 底背驰：价格创新低，MACD力度减弱
- 强背驰：力度减弱超过50%
- 弱背驰：力度减弱20-50%
```

#### 金叉/死叉
```python
- 金叉：DIF从下向上突破DEA（买入信号）
- 死叉：DIF从上向下突破DEA（卖出信号）
```

## 缠论核心原则

系统严格遵循缠论理论体系：

1. **不臆测** - 结构未完成就说「未完成」，不强行定论
2. **不迎合** - 分析结论不因「希望市场涨/跌」而偏移
3. **不糊弄** - 有争议的结构标注「观察中」，不假装确定
4. **不甩锅** - 决策失误主动承认，不找借口
5. **不遗漏** - 关键风险必须上报，不得擅自忽略
6. **不造假** - 模拟盘结果如实记录，不美化
7. **不干扰** - 收到冷却信号后，立即停止生成信号

## 纪律要求

### 智能体自律
- 实事求是，数据缺失时明确说明
- 客观分析，不受情绪影响
- 风险控制，严格执行止损
- 先验后实，模拟盘验证后再考虑实盘

### 风控规则
- 单笔交易不得超过总资金的20%
- 连续亏损3次后暂停决策
- 胜率 < 45% 时降低仓位至50%
- 胜率 < 40% 时暂停决策，重新优化
