# 缠论多智能体分析系统 v7.0

基于 LangGraph 和大语言模型的缠论全自动多智能体市场分析系统。

## v7.0 更新内容 ⭐

### 新增功能

#### 1. 策略参数优化 ⭐
- **网格搜索优化**：遍历所有参数组合，找到最优参数
  - 适用于参数空间较小的情况
  - 保证找到全局最优解
- **随机搜索优化**：随机采样参数空间
  - 适用于参数空间较大的情况
  - 比网格搜索更高效
- **贝叶斯优化**：基于已有结果智能采样
  - 使用高斯过程模型预测性能
  - 迭代优化，逐步收敛到最优解
- **遗传算法优化**：模拟生物进化过程
  - 选择、交叉、变异操作
  - 适用于复杂参数空间

#### 2. 增强版回测系统 ⭐
- **完整的回测框架**：
  - 支持自定义信号生成器
  - 支持多空双向交易
  - 自动计算所有绩效指标
- **详细绩效指标**：
  - 总交易次数、盈利交易、亏损交易
  - 胜率、总收益率、年化收益率
  - 最大回撤、夏普比率、索提诺比率
  - 平均盈利、平均亏损、盈利因子
  - 总盈亏、最大连续亏损
  - 资金曲线
- **风险管理**：
  - 自动止损
  - 仓位控制
  - 最大回撤监控

#### 3. 移动止损策略 ⭐
- **动态止损调整**：
  - 多头：价格创新高时上移止损位
  - 空头：价格创新低时下移止损位
- **保护利润**：
  - 在价格向有利方向移动时锁定利润
  - 减少回撤，提高风险收益比
- **可配置参数**：
  - 移动止损百分比（默认5%）
  - 可根据策略特性调整

#### 4. 策略绩效评估 ⭐
- **多策略对比**：
  - 对比不同参数组合的表现
  - 可视化比较关键指标
- **最佳策略推荐**：
  - 基于夏普比率推荐最佳策略
  - 综合考虑收益和风险
- **详细分析报告**：
  - 参数组合
  - 收益率、夏普比率、最大回撤
  - 胜率、盈亏比等关键指标

## v6.0 更新内容 ⭐

### 新增功能

#### 1. 增强版中枢识别算法 ⭐
- **动态阈值调整**：根据价格波动范围动态调整中枢识别阈值
- **中枢状态判断**：区分中枢的不同状态（构建中/已完成/延伸中/突破中）
- **中枢强度计算**：综合评估中枢强度（0-1）
  - 重叠次数权重：30%
  - 线段数量权重：30%
  - 时间跨度权重：20%
  - 价格范围权重：20%
- **多维度分析**：考虑中枢的上沿（ZG）、下沿（ZD）、高点（GG）、低点（DD）

#### 2. 线段延伸判断 ⭐
- **延伸类型识别**：
  - 趋势延伸：突破前高/前低
  - 震荡延伸：价格范围缩小
  - 突破延伸：形成新的趋势
- **趋势强度计算**：评估线段延续趋势的强度（0-1）
  - 趋势持续性：40%
  - 价格创新能力：40%
  - 回调深度：20%
- **突破点位识别**：识别关键突破点
- **目标位预测**：基于斐波那契扩展预测下一个目标位

#### 3. 增强版背驰识别算法 ⭐
- **多重确认机制**：
  - MACD背驰
  - 价格动量背驰
  - 成交量背驰
- **背驰强度量化**：综合评估背驰强度（0-1）
  - 价格动量权重：30%
  - MACD背离权重：40%
  - 成交量背离权重：30%
- **背驰类型细分**：
  - 顶背驰：价格创新高但指标未创新高
  - 底背驰：价格创新低但指标未创新低
- **确认次数统计**：记录多重确认的背驰信号

#### 4. 趋势力度指标 ⭐
- **ADX（平均趋向指数）**：量化趋势强度
  - 0-25：弱趋势或无趋势
  - 25-50：中等趋势
  - 50-75：强趋势
  - 75-100：极强趋势
- **价格趋势强度**：使用线性回归斜率评估趋势
- **动量强度**：使用ROC（变化率）评估动量
- **波动率指标**：使用标准差评估波动性
- **成交量强度**：比较当前成交量与平均成交量
- **综合趋势力度**：加权计算综合趋势力度（0-1）
  - 价格趋势权重：30%
  - 动量权重：30%
  - 波动率权重：20%
  - 成交量分布权重：20%

### 算法对比

#### 中枢识别
| 指标 | 原始算法 | 增强版算法 |
|------|---------|-----------|
| 阈值设定 | 固定阈值 | 动态阈值 |
| 状态判断 | 无 | 4种状态（构建/完成/延伸/突破） |
| 强度计算 | 无 | 综合强度评分（0-1） |
| 多维度分析 | 基础 | 上沿/下沿/高点/低点 |

#### 背驰检测
| 指标 | 原始算法 | 增强版算法 |
|------|---------|-----------|
| 检测维度 | 单一（MACD） | 多重（MACD/动量/成交量） |
| 确认机制 | 无 | 多重确认（>=2个指标） |
| 强度量化 | 无 | 综合强度评分（0-1） |
| 背驰类型 | 基础 | 顶/底背驰 + 指标类型细分 |

#### 趋势分析
| 指标 | 原始算法 | 增强版算法 |
|------|---------|-----------|
| 趋势强度 | 无 | ADX + 综合力度（0-1） |
| 线段延伸 | 无 | 延伸类型 + 趋势强度 |
| 目标预测 | 无 | 基于斐波那契扩展预测 |

  - 固定百分比止损
  - 支撑/阻力位止损
  - ATR止损
  - 移动止损
- **仓位管理**：根据置信度和风险等级动态调整仓位
  - 单笔交易最大亏损限制（2%）
  - 单个品种最大仓位限制（30%）
  - 总最大仓位限制（80%）
- **回撤监控**：实时监控账户回撤，超过限制触发冷却
- **连续亏损保护**：连续亏损5次后自动触发冷却模式
- **风险评分**：综合评分系统（0-100），量化风险等级

#### 3. 实战理论智能体 ⭐
- **三类买卖点识别**：
  - 第一类买点：下跌趋势底背驰，形成底分型
  - 第二类买点：上涨回调不破前低，形成底分型
  - 第三类买点：向上突破中枢上沿
  - 第一类卖点：上涨趋势顶背驰，形成顶分型
  - 第二类卖点：下跌反弹不破前高，形成顶分型
  - 第三类卖点：向下突破中枢下沿
- **仓位管理建议**：根据信号置信度提供仓位建议
- **操作节奏把控**：提供入场/离场时机建议
- **分批交易策略**：分批入场、分批离场建议

#### 4. 风控智能体 ⭐
- **风控审核**：对所有交易决策进行风险审核
- **一票否决权**：可以拒绝任何风险过高的交易
- **风险评分**：为每个交易决策计算风险评分
- **风控建议**：提供具体的风控建议（止损价格、仓位大小）
- **否决条件**：
  - 超过最大回撤限制（15%）
  - 连续亏损次数过多（5次）
  - 今日交易次数达到上限（10次）
  - 今日亏损超过限制（5%）
  - 置信度过低（< 50%）
  - 系统处于冷却模式

### 风控原则

1. **本金保护第一** - 任何时候，保护本金比盈利更重要
2. **严格止损** - 止损是硬约束，不得随意调整
3. **仓位控制** - 单笔交易最大亏损不超过2%
4. **风险分散** - 不要把所有资金投入单一品种
5. **情绪控制** - 极端情绪下暂停交易
6. **纪律执行** - 风控规则必须严格执行，不得例外

### v3.0 - 辅助维度分析

#### 1. 市场情绪分析

#### 恐慌贪婪指数
- **指数范围**: 0-100
- **极度恐惧** (0-24): 市场恐慌，可能接近底部
- **恐惧** (25-44): 市场情绪悲观，可能存在机会
- **中性** (45-55): 市场情绪平稳
- **贪婪** (56-75): 市场情绪乐观，可能存在泡沫
- **极度贪婪** (76-100): 市场过热，可能接近顶部

#### 资金费率
- **正费率**: 多头占优，多头支付给空头
- **负费率**: 空头占优，空头支付给多头
- **零费率**: 多空平衡

#### 爆仓数据
- 多单爆仓数量 vs 空单爆仓数量
- 爆仓总金额
- 爆仓市场影响

#### 持仓量

## 决策优先级

1. **系统监控告警** - 最高优先级，任何告警暂停决策
2. **大级别结构** - 以大级别判断为准（权重30%）
3. **多维度共振** - 共振越多，置信度越高
4. **动力学确认** - 有背驰确认的信号优先（权重25%）
5. **市场情绪** - 极端情绪必须重视（权重15%）
6. **跨市场联动** - 宏观环境影响（权重15%）
7. **链上信号** - 基本面支撑（权重10%）
8. **模拟盘验证** - 胜率 < 45% 的信号降权（权重5%）
9. **风控否决** - 风控拥有一票否决权

## 缠论核心原则

系统严格遵循缠论理论体系：

1. **不臆测** - 结构未完成就说「未完成」，不强行定论
2. **不迎合** - 分析结论不因「希望市场涨/跌」而偏移
3. **不糊弄** - 有争议的结构标注「观察中」，不假装确定
4. **不甩锅** - 决策失误主动承认，不找借口
5. **不遗漏** - 关键风险必须上报，不得擅自忽略
6. **不造假** - 模拟盘结果如实记录，不美化
7. **不干扰** - 收到冷却信号后，立即停止生成信号
